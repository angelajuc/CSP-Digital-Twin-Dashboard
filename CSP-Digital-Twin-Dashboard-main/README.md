## CSP Digital Twin Dashboard for City of Marietta

![Demo gif](https://github.com/angelajuc/CSP-Digital-Twin-Dashboard/blob/5f844dde574f2ff4054ce6d26ad39dc864fadf53/dashboard-view.gif)

### Run it

    git clone https://github.com/angelajuc/CSP-Digital-Twin-Dashboard.git
    cd CSP-Digital-Twin-Dashboard/basic
    npm install
    npm start

    open http://localhost:3000/

### Access token

    mapboxgl.accessToken = // YOUR_API_KEY_HERE

Replace the [API key](https://docs.mapbox.com/help/getting-started/access-tokens/) in `src/Map.js`.

Runs the app from http://localhost:3000
